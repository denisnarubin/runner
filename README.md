# 3D-runner

Динамическая 3D-игра runner, реализованная на Three.js и TypeScript.
Проект оптимизирован дляPlayable Ads и собран в один автономный HTML-файл.

Игрок управляет персонажем, который автоматически бежит вперёд по платформе. Задача — собирать монеты, избегать бомб и проходить через ворота с модификаторами счёта. В конце уровня персонаж исполняет победный танец, после чего отображается экран с результатом.

1.  **Готовый файл:** Находится в папке `dist/index.html`. Его можно запустить локально в любом браузере.
2.  **Исходники:** Расположены в папке `src/`.


## 🛠 Как собрать и запустить

### 1. Клонирование репозитория
```bash
git clone https://github.com/denisnarubin/runner.git
#Переход в директорию проекта
cd playable-runner
```

### Запуск в режиме разработки:
```bash
#Установите зависимости: 
npm install
#Запуск:
npm run dev
```

### Сборка финального index.html:
```bash
#Выполните команду: 
npm run build
#Сборщик Vite с плагином `vite-plugin-singlefile` упакует все необходимое в файл `dist/index.html`.
```

## 🎮 Геймплей
* Бег: Персонаж движется вперёд автоматически
* Управление: Свайпы влево/вправо для смены полосы
* Монеты: +1 к счёту при сборе
* Бомбы: При столкновении — взрыв, падение персонажа, экран поражения
* Ворота: Модификаторы счёта (+5, -10, ×2)
* Финиш: Персонаж танцует, отображается итоговый счёт
